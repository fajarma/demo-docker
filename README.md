# demo-docker
Repo for docker exercise

### Install dependencies
```
npm install
```

### Run Docker
```
docker-compose up
```
or
```
docker-compose up -d
```
if you want run docker in background.

### See container list
```
docker ps
```

### Stop docker
```
ctrl + c
```
or
```
docker-compose stop
```
if you run docker in background
